### Farmhouse Booking App

A single-page farmhouse booking application built using **pure HTML5 with inline styling**.

### Features
- Booking form with relevant inputs
- Guest selection using a drag slider
- Live guest count display
- Side-by-side booking form and location map
- WhatsApp redirection for quick contact
- Google Maps embedded using iframe
